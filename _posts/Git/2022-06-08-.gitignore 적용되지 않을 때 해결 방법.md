### ignore 처리된 파일이 자꾸 commit되는 경우

이와 같은 경우에는 **git cache 문제**이기 때문에 터미널에서 **cache 삭제 후 다시 commit** 하면 된다!

```
git rm -r --cached .
git add .
git commit -m "clear git cache"
git push
```
